Ruby Classes
2/8/2016

Today I'm gonna teach you the basics about a Ruby Class. A Ruby Class allows us to define methods to be used for working on an object but we can create new instances of the object when we want to use different data. This new instance is also completely seperate from other objects made. I'll show an example here and then explain what is happening.

           Class Information

            def initialize(age, name, hometown)
              @age = age
              @name = name
              @hometown = hometown
            end

            def print_info
              puts "Hello #{@name}! You are #{@age} and you live in #{@hometown}."
              end
            end

            info = Information.new(25, "Kyle", "Vallejo")
            info.print_info

        Hello Kyle! You are 25 and you live in Vallejo.

So first you can see I defined my class to be class Information. Next you want to define an initiliaze to determine what variables are going to be used. For my information I want to have age, name, and hometown. This is where you start to see something new for our variables. I want to be able to use them in other methods in my class so I have to label them as an instance variable. In order to label something as an instance variable you just have to put an @ symbol in front of it. Now that we've gotten our variables set up we can move on to the next method. Here I want to print a string with the data that is given. By using #{@instance_variable} I can have the variable be called and then transformed into a string. So now I want to pass some data through to see if my class works. I assign info to be Information.new which tells Ruby to run the class and make a new instance. After I give it my values I want assigned to my variables. Finally after getting the info setup I can think call on my print_info method to run and get my string to print out. There's a lot more to classes but here is a good start to figure out some of the capabailities you have using one.
