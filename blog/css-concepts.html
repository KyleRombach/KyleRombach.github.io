CSS Concepts
1/17/2016

Today I'm going to discuss using Classes vs ID's in CSS. They both do the same thing however there is a certain time you should use each one.

First let's start with Classes. In order to write a class you would do element class="class name" in your HTML. Then in your CSS you would write .classname { styling inside here}. You can use class on multiple elements and have variations on the same element as well. These two reasons are why you want to use class when applying styling to to multiple things.

So now we know what Classes are used for let's talk about using ID's. ID's are very specific, in fact each element can only have one ID and your page can only have one element that has that ID. ID's are only to be used when you have to change one thing. You can't have multiple ID's within eachother.

Now that you guys have an idea of what a Class and ID is, hopefully you understand when you use each one. Just remember Classes are meant to be used on multiple objects and ID's are one use only.
