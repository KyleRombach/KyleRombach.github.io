Ruby Array
How to create an Array
ary = Array.new #=> []
Array.new(3) #=> [nil, nil, nil]
Array.new(3, true) #=> [true, true, true
Accessing elements in an Array
arr = [1, 2, 3, 4, 5, 6]
arr[2] #=> 3
arr[100] #=> nil
arr[-3] #=> 4
arr[2, 3] #=> [3, 4, 5]
arr[1..4] #=> [2, 3, 4, 5]
arr[1..-3] #=> [2, 3, 4]
Add element to end of Array
arr = [1, 2, 3, 4]
arr.push(5) #=> [1, 2, 3, 4, 5]
arr << 6 #=> [1, 2, 3, 4, 5, 6]
Add item to front of Array
arr.unshift(0) #=> [0, 1, 2, 3, 4, 5, 6]
Insert element anywhere in Array
arr.insert(3, 'apple') #=> [0, 1, 2, 'apple', 3, 4, 5, 6]
Remove last element from Array arr = [1, 2, 3, 4, 5, 6]
arr.pop #=> 6
arr #=> [1, 2, 3, 4, 5]
Retrieve and remove first element from Array
arr.shift #=> 1
arr #=> [2, 3, 4, 5]
Delete particular element from Array
arr = [1, 2, 2, 3]
arr.delete(2) #=> 2
arr #=> [1,3]
